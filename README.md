# HNO Skills

OpenClaw skills built by High Noon Office.

## Skills

### Ghost Publishing Pro
The complete Ghost CMS toolkit for OpenClaw agents. Migrate, automate, publish — without
touching the browser.

13 workflows: content creation, post management, subscriber ops, analytics, code injection,
full site migration. API-first with browser fallback.

→ [ClawHub](https://clawhub.com/@highnoonoffice)

### OC Brain Map
Visualize how your OpenClaw agent moves through its memory vault. Files become nodes,
co-access patterns become edges. A D3.js force-directed graph of actual agent cognition.

→ [ClawHub](https://clawhub.com/@highnoonoffice)

### Second Brain Visualizer
Drop ideas anywhere — Telegram, Slack, wherever you think — and surface the signal later.
D3.js node graph, categorized, tagged, explorable.

→ [ClawHub](https://clawhub.com/@highnoonoffice)

### Library of Babel
Bidirectional coordinate engine for Borges' infinite Library. Find any text. Read any page.
Deterministic — no randomness, no database.

→ [ClawHub](https://clawhub.com/@highnoonoffice)
